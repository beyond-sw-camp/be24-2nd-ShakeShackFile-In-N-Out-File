✨ 프로젝트 소개

FileInNOut은 단순한 클라우드 저장소를 넘어, 파일 관리 시스템과 실시간 문서 협업 기능을 완벽하게 통합한 차세대 워크스페이스입니다.
저장소와 편집 도구 사이를 오가는 번거로움을 줄이고, 팀의 생산성을 극대화합니다.

🎯 Problem → Solution

😵 Problem

파일 저장소와 협업 문서 툴이 분리되어 데이터가 파편화됨

폴더 구조 내에서 실시간 편집 중인 문서를 바로 확인하기 어려움

복잡한 파일 권한 관리와 공유 프로세스로 인한 협업 효율 저하

😎 Solution

MinIO 기반 오브젝트 스토리지로 안정적인 파일 관리 제공

Node.js & WebSocket을 이용한 지연 없는 실시간 동시 편집 에디터

블록 단위의 유연한 문서 구성 및 무한 계층 페이지 구조 지원

🏗️ 시스템 아키텍처 (Architecture)

*현재 구조는 **High Availability(고가용성)*와 트래픽 분산에 초점을 맞추고 있습니다.

Load Balancing: Nginx 및 HAProxy를 통한 클라이언트 요청 분산

Database Strategy: Master-Slave 구조의 MariaDB를 채택하여 쓰기 작업의 원본성 유지 및 읽기 성능 최적화

File Storage: MinIO 오브젝트 스토리지를 사용하여 정적 파일 서비스의 확장성 확보

Real-time Engine: Node.js 기반의 전용 서버로 실시간 동시성 보장

🚀 추가 제언 (부족한 부분)

실시간 편집 기술 상세: WebSocket 외에 CRDT나 OT 중 어떤 동시성 제어 방식을 사용하는지 기술하면 전문성이 높아집니다.
